CHEAT SHEET

HTML

Basic tags

  1. html - used as a root tag for the structure

  2. title - used to create the page title

  3. head- used for section headers

  4. body - creates section for the content

  5. p- creates section for individual paragraphs


CSS

Basic Syntax

  1. Seclectors
    selector { Property: value; }

    • Tag - used to interact with HTML tags

      • eg. p {color:red;}

    • Class- used to interact with the classes in HTML

      • eg. p class ="background"
      p.background {background-color:red;}


    • ID - used to interact with HTML id

      • eg. p {color:red;}

  2. color - used to change the color of the selector

    3. eg. p {color:red;}

  3. text - used to align text

    4. ex. text-align:center;
